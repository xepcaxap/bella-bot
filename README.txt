Инструкция по запуску Бэллы — маскота клана Para Bellum:

1. Установи зависимости:
   pip install python-telegram-bot openai python-dotenv

2. Создай файл `.env` в корне и вставь туда свои ключи:
   TELEGRAM_BOT_TOKEN=твой_токен_от_BotFather
   OPENAI_API_KEY=твой_ключ_OpenAI

3. Запусти файл:
   python main.py

4. Добавь бота в Telegram-группу и дай ему права на чтение сообщений и добавление новых участников.
